# 📘 03.cpp - C++ 기초 종합 예제 모음

이 프로젝트는 C++ 프로그래밍의 기초 문법을 종합적으로 실습할 수 있도록 구성된 예제 모음입니다.  
조건문, 반복문, 함수, 수학 계산 등 다양한 주제를 포함하고 있으며, 초급자들이 실습을 통해 익힐 수 있도록 구성되어 있습니다.

---

## 📂 파일 설명

- **파일명**: `03.cpp`
- **구성**: 70개 이상의 예제 코드 블록
- **주제**: 반복문, 조건문, 함수, 사용자 입력, 수학 계산, 제어문 등

---

## 🛠 주요 학습 내용

### 🔁 반복문 실습
- `for`, `while`, `do-while` 문을 활용한 합계 계산, 구구단 출력, 별 출력 등
- 중첩 반복문을 활용한 구구단 테이블, 조합 출력 등

### 🧠 조건문 실습
- `if`, `else if`, `switch-case`를 활용한 성적 평가, 음식 주문, 요금 계산
- 사용자 입력에 따라 동적으로 동작하는 프로그램 구조

### 🔣 함수 실습
- `void`형 함수와 `return`형 함수 모두 구현
- 매개변수 전달 방식 (`call by value`, `call by reference`) 실습
- 함수 내에서 다른 함수 호출하기 (`함수 호출의 재사용성` 학습)

### 🔢 수학/계산 관련
- 제곱근 계산 (sqrt 함수 vs 뉴턴-랩슨 방법)
- 수열 합, 판별식 계산, 제곱 함수 (`pow`) 활용
- 사용자 입력 기반 수학 문제 풀이

### 📌 기타 주요 예제
- 달력 출력
- 소수 판별
- 피타고라스 삼각형 높이 계산 (삼각함수)
- 음식 주문 시 누적 금액 계산
- 거리 계산기, 원/삼각형/사각형 넓이 계산 등 실용적 예제 다수 포함
