# 🚀 C++ 기초 학습 및 연산, 조건문 실습 📚

이 프로젝트는 C++ 언어의 기초를 학습하면서 다양한 조건문과 연산자 사용법을 실습한 내용을 담고 있습니다. 여러가지 간단한 프로그램을 통해 개념을 익히고 실습했습니다.

## 📂 프로젝트 구성

* **02.cpp** : 다양한 기초 연산 및 조건문을 활용한 예제 코드 모음

---

## 📝 학습한 내용

### 📌 기본 연산

* 입력받은 숫자의 크기 비교 및 최대값 찾기
* 자료형 변환을 통한 수학적 연산
* 절댓값 구하기
* 홀수, 짝수 판별

### 📌 조건문 및 switch-case 사용법

* 점수에 따른 학점 판정 프로그램
* 알파벳 문자 입력 후 모음 또는 자음 판별
* 간단한 연산 결과 출력
* 평균 점수 및 표준 체중 계산
* 화씨를 섭씨로 변환
* 근의 공식 활용하여 2차 방정식의 근 구하기
* 연도 윤년 판단
* BMI에 따른 체중 판정
* 전력요금 계산

### 📌 입력값 처리 및 조건적 출력

* 입력받은 수의 자릿수 별 출력
* 시간 및 거리 변환
* 성적 및 급여 관련 세금 계산
* 특정 조건에 따라 다르게 입력 처리 및 결과 출력

### 📌 반복문 및 goto문 활용

* 반복문과 goto문을 사용한 합계 계산
* goto문을 활용한 입력값의 평균 계산

```bash

```

```bash
program.exe
```
