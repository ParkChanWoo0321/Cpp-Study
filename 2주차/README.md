# 📘 02.cpp - C++ 기본 문법 실습 종합

이 파일(`02.cpp`)은 **C++ 언어의 기초 문법 학습**을 위한 종합 예제 파일로, 입력/출력, 조건문, 반복문, 수학 계산, 자료형 변환, 다양한 분기처리, 함수 활용 없이 모든 기초 문법을 직접 실습할 수 있도록 구성되어 있습니다.

---

## 🔧 주요 학습 주제 및 내용 구성

### 📥 입력 / 출력

* `cin`과 `cout`을 통한 사용자 입력 및 출력 형식
* `endl`, `\n` 활용법

### 🔢 변수 및 자료형

* `int`, `double`, `char` 등 다양한 자료형 사용
* 명시적 형변환 `(int)`, `(double)`을 통한 계산 정확도 확보

### 📐 연산자

* 사칙연산, 나머지, 복합 대입, 논리 연산자 등 다양하게 활용
* `삼항 연산자`를 통한 간단한 조건 비교 및 대입

### 🔁 반복문 및 분기

* `if`, `else if`, `else` 조건문
* `switch-case` 분기 처리
* `goto`문을 활용한 루프 구조 실습 (권장되진 않지만 학습용으로 포함)

### 📊 계산 예제 포함 항목

* 평균 계산, 속도, 연비, 체중 계산, 단위 환산 등 다양한 실생활 예제
* BMI 계산을 통한 조건 분기
* 전기요금, 세금, 급여 등 실용 수식 적용

### 🔠 문자 / 문자열 처리

* `char` 입력 및 알파벳 구분 (모음/자음, 대소문자, 숫자, 특수문자 등)
* `switch`와 조건문을 사용한 분기 처리

### 📏 단위 변환 실습

* cm <-> inch, feet, lb, kg 등 단위 변환 공식
* 섭씨 <-> 화씨 변환

### 💡 다항식 계산, 2차 방정식 근 구하기

* 근의 공식 적용 (`sqrt` 함수 활용)
* 복잡한 수식 안에서 `조건문`과 `연산자` 결합 활용

### 📌 기타

* 각 자리 숫자 추출 (천의 자리, 백의 자리 등)
* 윤년 판별, 사분면 판별, 학점 부여 시스템 등 다양한 분기 조건 실습

---

## 🖥️ 실행 환경

* Microsoft Visual Studio 또는 Dev-C++
* 또는 CLI 환경에서 실행:

```bash
> g++ 02.cpp -o main
> ./main
```
