# 📘 09.cpp - C++ 함수 중복, 기본 매개변수, 클래스 정적 멤버 실습 정리

이 파일은 C++에서 함수 오버로딩, 기본 인자값, 정적 멤버 변수, 동적 메모리, 클래스 기본 생성자와 소멸자 등을 종합적으로 실습한 학습 예제 모음입니다.

---

## ✅ 주요 학습 내용

### 🔁 함수 오버로딩 & 기본 매개변수
- `big(int a, int b)` vs `big(int* p, int size)` : 같은 이름의 함수지만 인자 타입이 다름
- 함수 기본 매개변수를 활용하여 인자가 없거나 일부만 있을 때도 유연하게 호출 가능
- `void star(int a = 5)`, `void msg(int id, string text = "")` 등

### 📦 클래스와 정적(static) 멤버
- `Person` 클래스에서 `static int sharedMoney` 사용
- 클래스 간 공유되는 정적 자원의 선언/정의/접근 방법 이해
- 객체마다 다른 money, 하나의 공유 자원 sharedMoney

### 🧹 소멸자 활용과 객체 수 관리
- `Circle` 클래스에서 생성자/소멸자 사용해 객체 개수 추적
- `static int numOfCircles`로 현재 생존한 객체 개수 관리

### 📚 문자열 및 배열 관련 실습
- 문자열 입력 후 사전순 첫/끝 단어 찾기
- 문자열을 한 글자씩 왼쪽으로 shift 하여 출력
- 배열을 인자로 넘기고 합산 처리 (`int add(int a[], int size, int b[] = {})`)

### 👤 클래스와 기본 인자 결합
- `Person` 클래스에서 `id`, `name`, `weight` 기본값 지정
- 다양한 인자 조합으로 생성자 호출 → `Person grace`, `Person(2, "Ashley")`, etc.

### 🧠 기타 개념 정리
- `new`, `delete[]` 통한 동적 배열 할당 및 해제
- `this->` 키워드로 클래스 멤버 명확하게 지정
- 중첩 조건문을 활용한 `big()` 함수 구성

---

## 🧪 실습 예제 포함 함수/클래스 목록

- `big()` 함수: 오버로딩 + 기본 인자값
- `add()`: 배열 합산 함수
- `Person` 클래스 (정적 멤버 & 기본값 생성자)
- `Circle` 클래스 (객체 수 추적용 정적 멤버)
- `f()`, `star()`, `msg()` 함수: 출력용 실습
