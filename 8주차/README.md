# 📘 08.cpp – C++ 포인터·배열·동적메모리·클래스 종합 실습 코드

`08.cpp`는 C++에서 **클래스, 포인터, 배열, 이중배열, 동적 메모리, 소멸자, 객체 배열, 자판기 시뮬레이터**까지 전 범위를 실습한 종합 예제 모음입니다.  
각 예제는 독립적으로 구성되어 있으며, C++ OOP 기본기와 포인터 활용 능력을 강화하는 데 목적이 있습니다.

---

## ✅ 주요 학습 내용 요약

### 1. 클래스 포인터 사용
- 객체 주소를 포인터가 가리키고 `->` 연산자로 멤버 접근
- `(*p).getArea()` 형태의 역참조 방식도 함께 실습

### 2. 배열과 포인터를 이용한 클래스 객체 순회
- `Circle circleArray[3]` 형태의 객체 배열 생성
- 포인터를 배열 첫 번째 원소에 연결하여 반복문으로 순회

### 3. 생성자를 가진 클래스 배열 초기화
- `{ Circle(10), Circle(20), Circle() }` 형태의 배열 초기화 문법 학습

### 4. 객체 배열 + 포인터 활용 출력
- `Color` 클래스 배열을 만들어 색상값을 순서대로 출력
- 포인터 증가 연산을 통해 배열 순회

### 5. 클래스의 이중배열(2차원 배열)
- `Circle circles[2][3]` 2D 배열 생성
- 각 위치에 접근하여 Radius 설정 및 면적 출력

### 6. 생성자/소멸자 실행 순서 확인
- 객체 배열 초기화 시 생성자 호출 순서 출력
- 프로그램 종료 시 소멸자 실행 확인

### 7. 동적 메모리 할당 (`new`, `delete`)
- `int* p = new int[n]` 으로 정수 배열 동적 생성
- 입력된 정수들의 평균값 계산 후 출력
- 메모리 부족시 예외 처리 포함

### 8. new를 이용한 객체 생성
- `Circle* p = new Circle(radius);`
- 사용 완료 후 `delete p` 로 해제

### 9. 가변 입력값에서 최대값 찾기 클래스
- `Sample` 클래스 내부에서
  - 동적 배열 생성
  - 사용자 입력
  - 전체 출력
  - 가장 큰 수 계산

### 10. 커피 자판기 시뮬레이터 클래스
- `Container` / `CoffeeVendingMachine` 두 개의 클래스 구조
- 커피, 물, 설탕 용기 3개 관리
- 메뉴:
  - 에스프레소
  - 아메리카노
  - 설탕커피
  - 잔량보기
  - 원료 채우기
- while 루프 기반 무한 동작 구조

| 클래스 | 용도 |
