## 1주차 (01.cpp) 🌱 입출력 · 자료형 · 문자열 기초

- 🖨️ `cout` / ⌨️ `cin` 기본 입출력, `endl`, `\n`, `\t`
- 🔢 `int`, `double`, `char` 변수 선언/연산
- 🔤 ASCII 코드 이해 + 형변환 `(int)`, `(char)`로 대/소문자 변환
- 🧵 C 문자열: `char[]`, `strlen`, `strcmp`, `sizeof`
- 🔁 `while(true)` + `break` 등 반복/조건 기본 흐름
- 🧮 단위변환/요금/급여/합·평균 등 기초 계산 문제 풀이

## 2주차 (02.cpp) 🧩 조건문 · switch · 삼항 연산자 · 형변환 심화

- ✅ `if / else if / else`로 구간 분기(학점/할인/요금)
- 🎛️ `switch(score/10)` 같은 패턴으로 분기 처리
- ❓ 삼항 연산자 `?:`로 최댓값, 사분면 판정 등
- 🔄 정수/실수 연산 차이(캐스팅 후 나눗셈/나머지 연산)
- 🪙 동전 분해(500/100/50/10/1), 🔢 자리수 분해
- 🗓️ 윤년 판정(`&&`, `||`) 등 논리 연산 조합
- 🧷 `goto`를 이용한 반복 흐름 실습(합/평균/팩토리얼 등)

## 3주차 (03.cpp) 🔁 반복문 · 중첩루프 · 난수 · 함수 분리

- 🔂 `for`, `while`, 무한루프 + `break/continue`
- 🧮 누적 합/수열/팩토리얼/거듭제곱 계산
- ✖️ 중첩 반복문(2중/3중): 구구단 표, 조합 탐색
- 📅 `setw()`로 표/달력 형태 정렬 출력
- 🎲 `srand(time(NULL))`, `rand()` 난수 생성 + `*`로 시각화
- ⌨️ `_getch()`로 단계별 진행 제어
- 🧱 기능별 함수화(거리/배수/소수/근의 공식 등) + 참조 전달 연습

## 4주차 (04.cpp) 🧠 스코프 · 포인터 · 배열/정렬 · static · swap

- 📦 블록 스코프 & 변수 가려짐(Shadowing)
- 👉 포인터 기초: `&`, `*`, 포인터로 값 변경
- 🧺 배열과 포인터 관계: `a == &a[0]`, `*(a+i)` 접근
- 🔎 최댓값/최솟값 + 위치(index) 찾기
- 🔃 정렬(중첩 for + swap) 구현
- 🌍 전역/지역 변수 차이 확인
- 🧊 `static` 지역 변수의 값 유지 특성 확인
- 🔁 swap 3종 비교
  - 값 전달(변화 없음) / 포인터 전달 / 참조 전달
- 📝 2차원 배열 채점 + 정렬로 석차 출력

## 5주차 (05.cpp) 🧮 2차원 배열 포인터 · 행렬 · 통계/빈도 · 탐색/정렬

- 🧱 2차원 배열 메모리 구조와 포인터 접근: `*(*(a+i)+j)`
- 🧾 배열 포인터: `int (*p)[3]` 활용
- ➕ 행렬 덧셈 / ✖️ 행렬 곱 / 🔁 전치행렬
- 📊 행/열 합 + 전체 합을 마지막 행/열에 누적
- 📈 평균/분산/표준편차, 빈도수(freq), 히스토그램(`*`)
- 🔤 랜덤 대문자 생성 + 빈도 카운트
- 🔍 순차 탐색 & 🔎 이진 탐색(정렬 후)
- 🧩 배열/행렬 기능을 함수로 분리(포인터/배열 인자)

## 6주차 (06.cpp) 🪪 문자열 포인터 · 함수 포인터 · 구조체

## 7주차 (07.cpp) 🏗️ 구조체 포인터 심화 + 클래스(OOP) 입문

- 🧾 구조체 배열을 포인터로 처리 `(p+i)->field`
- 🧱 클래스 기본 구조: `private/public`, 멤버 함수
- 🧰 생성자 오버로딩(기본/매개변수/단일 인자)
- ✅ 메서드로 기능 구현(get/set, 계산/판정)
- ☕ `Account`, `CoffeeMachine` 같은 미니 클래스 설계
- 💥 소멸자(`~Oval`)로 객체 생명주기 확인

## 8주차 (08.cpp) 🧷 클래스 포인터 · 객체 배열 · new/delete · 합성

- 👉 클래스 포인터: `p->method()`, `(*p).method()`
- 🧺 객체 배열(1차원/2차원) 생성 및 포인터 순회(`p++`)
- 🔄 생성자/소멸자 호출 순서 출력으로 확인
- 🧠 동적 메모리
  - `new int[n]` / `delete[]`
  - `new Circle` / `delete`
- 🧱 클래스가 동적 배열을 멤버로 가지는 구조(Sample)
- 🧩 합성(Composition): `Container` + `CoffeeVendingMachine` 자판기 구현

## 9주차 (09.cpp) 🧵 string · 오버로딩 · 디폴트 인자 · static 멤버

- ✍️ `string` + `getline()`로 문자열 입력
- 🔤 사전순 비교로 가장 앞/뒤 문자열 찾기
- 🔁 `substr()`로 문자열 회전 출력
- 🧩 함수 오버로딩 `big(int,int)` / `big(int*,size)`
- 🎯 디폴트 인자(default parameter)로 선택적 호출
- 👥 `static` 멤버 변수/함수(공유 자원 개념)
- 🔢 static 카운터 + 소멸자: 객체 개수 추적(`new[]/delete[]` 포함)
- 🧑 생성자 디폴트 값으로 다양한 생성 방식

## 10주차 (10.cpp) 🧬 상속 · protected · 생성자 연결 · 다중 상속 · 스택

- 🧬 상속 기본: 부모 기능 재사용(`ColorPoint : Point`)
- 🔒 `protected` 멤버: 자식 클래스에서 접근 가능
- 🔗 초기화 리스트로 부모 생성자 호출(`: Point(x,y)`)
- 🧬 다단계 상속(`TV -> WideTV -> SmartTV`)
- 🧩 다중 상속(`Calculator : Adder, Subtractor`)
- 📦 상속 기반 자료구조:
  - `BaseArray`(동적 배열 관리)
  - `MyStack`(push/pop/용량/크기)

## 11주차 (11.cpp) 🎭 virtual · 다형성 · 가상 소멸자 · 추상 클래스

- 🧠 `virtual` 유무에 따른 호출 차이(정적/동적 바인딩)
- 🪄 업캐스팅: `Shape* = new Circle()` 형태로 다형성 사용
- 🧬 다단계 상속에서도 오버라이딩 동작 확인
- 🧷 `Shape::draw()`처럼 부모 함수 직접 호출
- ⚰️ 가상 소멸자: `Base*`로 delete 시 안전한 소멸 순서
- 🧩 추상 클래스/순수 가상 함수(`=0`)로 인터페이스 설계
- 🧱 run() 흐름 고정 + calc/convert만 구현하는 패턴 실습

## 12주차 (12.cpp) 🧩 인터페이스 설계 + 다형성 배열 + 동적 할당

- 🚦 논리 게이트 인터페이스
  - `AbstractGate` + `operation()=0`
  - AND/OR/XOR 구현 → 같은 함수명, 다른 동작(다형성)
- 🧺 `Shape* p[3]`에 서로 다른 도형 객체를 `new`로 저장
- 📐 `getName() / getArea()`로 공통 인터페이스 사용
- 🧼 `delete`로 메모리 해제
- ✅ “기본 구현(0 반환)” → “완전 추상 클래스(무조건 구현)” 구조로 개선
