# C++ 1주차 ~ 12주차 학습 정리 📌

---

## 1주차 (01.cpp) 🌱 입출력 · 자료형 · 문자열 기초
- `cout`, `cin`, `endl`, `\n`, `\t` 기본 입출력
- `int / double / char` 자료형과 산술 연산
- ASCII 코드 + 형변환으로 대/소문자 변환
- C 문자열(`char[]`, `strlen`, `strcmp`, `sizeof`)
- 반복/조건 기본(`while`, `break`)
- 단위 변환·합/평균·요금 계산 등 기초 문제 풀이

---

## 2주차 (02.cpp) 🧩 조건문 · switch · 삼항 연산자 · 형변환
- 최댓값/절댓값/짝홀 등 조건 판정
- `if-else if` 구간 분기(학점/할인/요금)
- `switch-case`로 분기 처리(점수·문자 판별)
- 정수/실수 연산 차이(캐스팅 후 나눗셈/나머지)
- 자리수/동전 분해 문제
- 논리 연산 조합(윤년 판정 등)
- `goto`로 반복 흐름 실습(합/평균/팩토리얼)

---

## 3주차 (03.cpp) 🔁 반복문 · 중첩루프 · 난수 · 함수 분리
- `for / while` 반복, 무한루프 + `break/continue`
- 중첩 반복문(구구단 표, 조합 탐색)
- 누적합/수열/팩토리얼/거듭제곱 계산
- `setw()`로 표/달력 정렬 출력
- `srand(time(NULL))`, `rand()` 난수 + `*` 시각화
- 기능별 함수화(거리/배수/소수/근의 공식 등), 참조 전달 연습

---

## 4주차 (04.cpp) 🧠 스코프 · 포인터 · 배열/정렬 · static · swap
- 블록 스코프와 변수 가려짐(Shadowing)
- 포인터 기본(`&`, `*`)과 포인터로 값 변경
- 배열-포인터 관계(`a == &a[0]`, `*(a+i)` 접근)
- 최댓값/최솟값 + 위치(index) 찾기
- 정렬(중첩 for + swap)
- 전역/지역 변수 차이 확인
- `static` 지역 변수의 값 유지 특성
- swap 3종 비교: 값/포인터/참조 전달
- 2차원 배열 채점 + 정렬로 석차 출력

---

## 5주차 (05.cpp) 🧮 2차원 배열 포인터 · 행렬 · 통계/빈도 · 탐색/정렬
- 2차원 배열 주소 구조 + 포인터 접근 `*(*(a+i)+j)`
- 배열 포인터 `int (*p)[3]` 활용
- 행렬 연산: 덧셈/곱/전치, 행·열·전체 합 누적
- 평균/분산/표준편차, 빈도수 배열(freq), 히스토그램
- 랜덤 대문자 생성 + 빈도 카운트
- 순차 탐색, 정렬 후 이진 탐색
- 배열/행렬 연산을 함수로 분리(포인터/배열 매개변수)

---

## 6주차 (06.cpp) 🧾 문자열 포인터 · 함수 포인터 · 구조체
- `char[]` vs `const char*`(리터럴 포인터) 차이
- 함수 포인터 `void(*pf)(void)`로 함수 선택/호출
- 구조체 `namecard`: 입력/출력, 초기화, 복사 대입, 배열 관리
- 구조체 반환 함수, 구조체 포인터 `->` 사용
- `sizeof(구조체)` vs `sizeof(포인터)` 비교
- 구조체 기반 성적표(합계/평균/학점/석차)

---

## 7주차 (07.cpp) 🏗️ 구조체 포인터 심화 + 클래스(OOP) 입문
- 구조체 배열을 포인터로 처리 `(p+i)->field`
- 클래스 기본 구조(`private/public`)와 멤버 함수
- 생성자 오버로딩(기본/매개변수/단일 인자)
- `Rectangle.isSquare()`, `Account`, `CoffeeMachine` 등 설계 실습
- 소멸자(`~Oval`)로 객체 생명주기 확인

---

## 8주차 (08.cpp) 🧷 클래스 포인터 · 객체 배열 · 동적 메모리 · 합성
- 클래스 포인터(`p->method`, `(*p).method`)
- 객체 배열(1D/2D) 생성 및 포인터 순회(`p++`)
- 생성자/소멸자 호출 흐름 확인
- 동적 메모리: `new/delete`, `new[]/delete[]`
- 동적 배열을 멤버로 가진 클래스(Sample)
- 합성(Composition): `Container` 기반 커피 자판기 구현

---

## 9주차 (09.cpp) ✨ string · 오버로딩 · 디폴트 인자 · static 멤버
- `string` + `getline()` 입력, 사전순 비교(최소/최대 문자열)
- `substr()`로 문자열 회전 출력
- 함수 오버로딩 `big(int,int)` / `big(int*,size)`
- 디폴트 인자(default parameter) 사용
- static 멤버(공유 변수/함수)와 객체별 멤버 비교
- static 카운터 + 소멸자로 객체 개수 추적(`new[]/delete[]`)
- 생성자 디폴트 값으로 다양한 생성 방식

---

## 10주차 (10.cpp) 🧬 상속 · protected · 생성자 연결 · 다중 상속 · 스택
- 상속 기본(부모 기능 재사용)
- `protected` 멤버 접근(자식에서 직접 사용 가능)
- 초기화 리스트로 부모 생성자 호출(`: Base(...)`)
- 다단계 상속(`TV -> WideTV -> SmartTV`)
- 다중 상속(`Calculator : Adder, Subtractor`)
- 상속 기반 스택 구현
  - `BaseArray`(동적 배열 관리)
  - `MyStack`(push/pop/용량/길이)

---

## 11주차 (11.cpp) 🎭 virtual · 다형성 · 가상 소멸자 · 추상 클래스
- `virtual` 유무에 따른 호출 차이(정적/동적 바인딩)
- 업캐스팅: `Shape* = new Circle()` 형태 다형성
- 오버라이딩 + 부모 함수 호출(`Shape::draw()`)
- 가상 소멸자: `Base*`로 delete 시 안전한 소멸 확인
- 추상 클래스/순수 가상 함수(`=0`)로 인터페이스 설계
- `run()` 흐름 고정 + 핵심 계산만 오버라이딩하는 구조 실습

---

## 12주차 (12.cpp) ✅ 인터페이스 설계 + 다형성 배열 + 동적 할당
- `AbstractGate` + `operation()=0`로 논리 게이트 인터페이스 설계
- AND/OR/XOR 구현으로 다형성 확인
- `Shape* p[3]`에 서로 다른 도형 객체를 `new`로 저장
- 공통 인터페이스(`getName/getArea`)로 호출 후 `delete` 해제
- 기본 구현에서 “완전 추상 클래스” 구조로 개선

---

## 전체 흐름 요약 🗺️
기초 문법 → 조건/반복 → 함수/포인터/배열(1D·2D) → 구조체 → 클래스(OOP) → 동적 메모리 → 상속 → 가상함수/다형성 → 추상 클래스/인터페이스 설계
