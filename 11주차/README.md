# 🧠 11.cpp - C++ 상속, 가상함수, 추상 클래스 실습

**11.cpp** 파일은 C++ 객체지향 프로그래밍의 고급 개념을 집중적으로 다룬 학습용 예제 모음입니다. 특히 **상속**, **오버라이딩**, **가상함수(Virtual Function)**, **소멸자**, **추상 클래스(Abstract Class)**, **다형성(Polymorphism)** 등의 개념을 직접 코드로 구현해보며 이해할 수 있도록 구성되어 있습니다.

---

## ✅ 주요 학습 내용 요약

### 📌 클래스 상속 & 함수 오버라이딩
- 동일한 함수 이름을 상속받은 클래스에서 재정의할 수 있으며, 부모 타입 포인터를 통해 호출 시 어떤 함수가 실행되는지 비교 학습
- `virtual` 키워드 유무에 따른 결과 차이 확인

### 🧱 가상 함수와 동적 바인딩
- 가상함수 선언 시 `Base*` 포인터가 가리키는 실제 객체의 `f()`가 실행됨
- 다양한 클래스 (`Base`, `Derived`, `GrandDerived`)에서 `f()`를 각각 오버라이딩

### 📐 가상 소멸자
- 부모 클래스에 가상 소멸자를 선언하지 않으면 자식 소멸자가 호출되지 않는 문제 해결
- `virtual ~Base()`로 안전한 리소스 정리 구조 구성

### 🔁 추상 클래스와 순수가상함수
- `virtual int func() = 0;` 형태로 순수가상함수 선언
- `Calculator`, `Converter`, `LoopAdder` 등은 인스턴스화할 수 없는 추상 클래스
- `GoodCalc`, `WonToDollar`, `KmToMile`, `ForLoopAdder` 등이 이를 구현하여 실사용

### 🧮 실용 예제 구현
- 계산기 클래스 (`add`, `subtract`, `average`) → 다형성 기반 사용
- 루프 기반 덧셈기 (`ForLoopAdder`) → 추상 클래스 구현
- 거리 및 통화 변환기 (`KmToMile`, `WonToDollar`) → 입력값 받아 계산 후 결과 출력

## 🧪 실습 클래스 목록

| 클래스 이름       | 설명 |
|------------------|------|
| `Base`, `Derived`, `GrandDerived` | 상속 및 오버라이딩 비교 실습 |
| `Shape`, `Circle` | 가상함수 기반 렌더링 구현 |
| `Calculator`, `GoodCalc` | 추상 클래스 기반 계산기 |
| `Converter`, `WonToDollar`, `KmToMile` | 단위 변환기 |
| `LoopAdder`, `ForLoopAdder` | 입력값 범위 합산 클래스 |

---

## 🧠 학습 포인트 정리

- `virtual` 함수가 있는 클래스는 반드시 소멸자도 `virtual`로 선언
- 추상 클래스는 직접 객체 생성 불가. 순수가상함수를 모두 구현한 파생 클래스만 인스턴스화 가능
- 포인터(`Base*`, `Shape*`)로 다양한 파생 객체를 가리킬 수 있음 → **다형성(Polymorphism)** 핵심
