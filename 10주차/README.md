# 🧱 C++ 상속과 클래스 활용 예제 (`10.cpp`)

이 코드는 C++에서 **클래스 상속**과 **protected/private 접근제한자**, **생성자/소멸자**, 그리고 **클래스 기반 스택 구현**까지 객체지향 프로그래밍의 핵심 개념을 실제 코드로 구현하며 학습하기 위한 예제 모음입니다.

단일 파일 `10.cpp`에 여러 예제를 순차적으로 작성하여 다양한 상속 구조와 클래스 간 관계를 이해할 수 있도록 구성되어 있습니다.

---

## 📚 주요 학습 내용

### ✅ 클래스 상속과 메서드 활용
- 기본 클래스 `Point`를 상속받는 `ColorPoint` 구현
- `this->` 키워드로 멤버 변수 구분
- 상속받은 부모 클래스의 멤버 함수 호출 (`showPoint()` 등)
- `protected`로 선언된 멤버 접근 테스트

### ✅ 다중 상속 및 접근 제어
- `Adder`, `Subtractor` 두 클래스를 상속한 `Calculator` 클래스 구현
- 각각의 연산을 자식 클래스에서 `calc()` 메서드로 통합 처리
- `protected` 멤버 함수는 자식 클래스 내부에서만 접근 가능함을 확인

### ✅ 생성자/초기화 리스트 및 기본 매개변수
- 생성자에서 `:Point(x, y)`와 같은 초기화 리스트 사용
- 디폴트 매개변수를 이용한 기본 생성자 구현

### ✅ 상속 체인 구조 구현
- 조상 클래스 `TV` → 자식 `WideTV` → 손자 `SmartTV`까지 계층적 상속
- 상속된 모든 클래스의 메서드를 하위 클래스 인스턴스에서 호출 가능

### ✅ 클래스 기반 스택 구현
- `BaseArray`: 기본 배열 메모리 관리용 부모 클래스
  - 배열 생성/해제, 내부 접근자(get/put), 용량(capacity) 제공
- `MyStack`: `BaseArray`를 상속받아 스택 기능 구현
  - `push()`, `pop()`, `length()` 메서드로 동작
  - `topIndex`를 내부적으로 관리
  - 오버플로우 / 언더플로우 검사 포함
